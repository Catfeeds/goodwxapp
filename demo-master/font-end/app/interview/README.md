## 项目举例

```
青芒fm(一个音乐平台)
技术栈：
1.pc管理后台+ h5前端+ios前端+ Androi前端。
2.后端用的技术是linux+php(lanavel模板)+mysql。
3.前端用的技术我这里只讲h5和管理后台。
3.1.h5移动端用vue写的，vue-cli搭架子，vue+vue-router+vuex生态。
（webpack+es6+模块化+组件化+数据驱动等）
3.2.管理后台用metronic前端模板做的（基于jquery+bootstrap）。

要做前面那个项目的基础：
1.pc端布局和移动端布局。（html+css）
2.页面效果和数据对接。（js+ajax+json）
3.学习jquery + bootstrap + vue。（这个前期不是重点，重点还是基础）
```
